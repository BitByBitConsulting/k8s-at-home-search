kube-prometheus-stack helm

kube-prometheus-stack is a Helm collection that deploys:

  • Grafana for dashboarding
  • Prometheus as metric database, together with an operator to manage it.
  • kube-state-metrics for exposing k8s metrics
  • prometheus-node-exporter for exposing node metrics
  • alertmanager for setting up alerts based on metrics or logs

Overall, kube-prometheus-stack is comperehensive and heavily configures all the tools.
