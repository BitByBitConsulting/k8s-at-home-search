immich-machine-learning helm

Immich is a self-hosted photo and video backup solution.

For immich, there are multiple helm deployments:

  • immich-web
  • immich-server
  • immich-typesense
  • immich-machine-learning (this one)
  • immich-microservices

For redis and postgres, there are multiple helm charts:

  • Redis: app-template/redis, bitnami/redis
  • PostgreSQL: operator cloudnative-pg/cloudnative-pg, bitnami bitnami/postgresql

Install

Install with:

helm repo add bjw-s oci://ghcr.io/bjw-s/helm/
helm install immich-machine-learning bjw-s/app-template -f values.yaml
